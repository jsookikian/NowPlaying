Data Structure Documentation

struct _gnsdk_musicidstream_callbacks_s

Structure of callback functions for status updates as various MusicID-Stream operations are performed.

Ensure that callback functions that access the same global variable(s) use serialized access methods to the shared variable(s).

Typedef Documentation

typedef struct _gnsdk_musicidstream_callbacks_s gnsdk_musicidstream_callbacks_t

Structure of callback functions for status updates as various MusicID-Stream operations are performed.

Ensure that callback functions that access the same global variable(s) use serialized access methods to the shared variable(s).

typedef gnsdk_void_t(GNSDK_CALLBACK_API * gnsdk_musicidstream_completed_with_error_callback_fn)(gnsdk_void_t *callback_data, gnsdk_musicidstream_channel_handle_t p_musicidstream_channel_handle, const gnsdk_error_info_t *p_error_info)

Callback function declaration for an error occurs during identification stage in MusicID-Stream.

Parameters
callback_data[in] Value passed to the user_data parameter of the function this callback was passed to
p_musicidstream_channel_handle[in] MusicID-Stream channel handle that the callback operates on
p_error_info[out] Final error value of MusicID-Stream operation
typedef gnsdk_void_t(GNSDK_CALLBACK_API * gnsdk_musicidstream_identifying_status_callback_fn)(gnsdk_void_t *callback_data, gnsdk_musicidstream_identifying_status_t status, gnsdk_bool_t *pb_abort)

Callback function declaration for MusicID-Stream channel identifying status.

Parameters
callback_data[in] Value passed to the user_data parameter of the function this callback was passed to
status[in] One of gnsdk_musicidstream_identifying_status_t values
p_abort[out] Set dereferenced value to GNSDK_TRUE to abort the operation that is calling the callback
typedef gnsdk_void_t(GNSDK_CALLBACK_API * gnsdk_musicidstream_processing_status_callback_fn)(gnsdk_void_t *callback_data, gnsdk_musicidstream_processing_status_t status, gnsdk_bool_t *pb_abort)

Note that at this time the classification and transition detection members of the enum gnsdk_musicidstream_processing_status_t are not available.

typedef gnsdk_void_t(GNSDK_CALLBACK_API * gnsdk_musicidstream_result_available_callback_fn)(gnsdk_void_t *callback_data, gnsdk_musicidstream_channel_handle_t p_musicidstream_channel_handle, gnsdk_gdo_handle_t response_gdo, gnsdk_bool_t *pb_abort)

Callback function declaration for a response available notification.

Parameters
callback_data[in] Value passed to the user_data parameter of the function this callback was passed to
gnsdk_musicidstream_channel_handle_t[in] MusicID-Stream channel handle that the callback operates on
response_gdo[in] GDO handle for the album response (of type GNSDK_GDO_TYPE_RESPONSE_ALBUM)
p_abort[out] Set the dereferenced value to GNSDK_TRUE to abort the operation that calls the callback
