#define GNSDK_GDO_CHILD_ALBUM

Retrieves a child album type.

Remarks:

Key is used as parameter for gnsdk_manager_gdo_child_get to retrieve child GNSDK_GDO_TYPE_ALBUM GDOs. The ordinal parameter for gnsdk_manager_gdo_child_get with this key indicates the n'th child album to retrieve. Album types are generally available from GNSDK_GDO_TYPE_RESPONSE_ALBUM GDOs.

#define GNSDK_GDO_CHILD_ALBUM_RESPONSE

Retrieves a child response album type.

Remarks:

Key is used as parameter for gnsdk_manager_gdo_child_get to retrieve child GNSDK_GDO_TYPE_RESPONSE_ALBUM GDOs. The ordinal parameter for gnsdk_manager_gdo_child_get with this key indicates the n'th child album to retrieve.

#define GNSDK_GDO_CHILD_AUDIO_WORK

Retrieves a child audio work type.

#define GNSDK_GDO_CHILD_MATCH_INFO
#define GNSDK_GDO_CHILD_TRACK

Retrieves a child track type.

Remarks:

Key is used as parameter for gnsdk_manager_gdo_child_get to retrieve child GNSDK_GDO_TYPE_TRACK GDOs. The ordinal parameter for gnsdk_manager_gdo_child_get with this key indicates the n'th child track to retrieve. Track types are generally available from GNSDK_GDO_TYPE_ALBUM GDOs.

#define GNSDK_GDO_CHILD_TRACK_BY_NUMBER

Retrieves a child track type matching the given track number.

Remarks:

Key is used as parameter for gnsdk_manager_gdo_child_get to retrieve child GNSDK_GDO_TYPE_TRACK GDOs. The ordinal parameter for gnsdk_manager_gdo_child_get with this key indicates the track number of the child track to retrieve.

Track types are generally available from GNSDK_GDO_TYPE_ALBUM GDOs.

#define GNSDK_GDO_CHILD_TRACK_MATCHED

Retrieves a child track type that was matched. GNSDK_GDO_CHILD_TRACK_MATCHED

Remarks:

Key is used as parameter for gnsdk_manager_gdo_child_get to retrieve child GNSDK_GDO_TYPE_TRACK GDOs. A child track is marked as matching for some queries that can match to a specific track. It is not always the case that there will be child track types marked as matching. Track types are generally available from GNSDK_GDO_TYPE_ALBUM GDOs.

#define GNSDK_GDO_VALUE_INPUT_TOC_ALBUM

Retrieves the TOC used to look up the album. Only present in albums retrieved by TOC lookups.

#define GNSDK_GDO_VALUE_USER_DATA

For albums coming out of a Supplemental Storage, this is the user data associated with the album when it was added to the store.
