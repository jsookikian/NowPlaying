gnsdk_fp_type_unknown 
gnsdk_fp_type_cx_query 

Indicates CX query fingerprint for identifying high quality audio files. Focuses on the first 15 seconds of a track. Required data handles: one.

gnsdk_fp_type_cx_reference 

Indicates The ‘reference’ version of the CX fingerprint. This is used for submission to Gracenote to allow for matching against query CX fingerprints. Required data handles: one.

gnsdk_fp_type_fapi_query_very_low_quality
