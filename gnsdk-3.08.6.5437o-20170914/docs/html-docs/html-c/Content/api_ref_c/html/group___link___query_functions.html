GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_link_query_content_count ( gnsdk_link_query_handle_t  link_query_handle,
gnsdk_link_content_type_t  content_type,
gnsdk_uint32_t *  p_count 
)

Retrieves count for the specified content that is described by the Link query handle options and content type flag.

Parameters
link_query_handle[in] Link query handle to retrieve content for
content_type[in] Type of content to count
p_count[out] Pointer to receive content count
Returns
LINKERR_NotInited - gnsdk_link_initialize was not successfully initiated
LINKERR_HandleObjectInvalid - Given link query handle is not a valid handle
LINKERR_HandleObjectWrongType - Given link query handle is not a valid link query handle
LINKERR_NotFound - Requested content is not available
LINKERR_NoError - Content count successfully retrieved

Remarks:

gnsdk_link_query_content_count can be called repeatedly on the same link query handle with different content type requests to retrieve the count for differing values of content type.

GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_link_query_content_retrieve ( gnsdk_link_query_handle_t  link_query_handle,
gnsdk_link_content_type_t  content_type,
gnsdk_uint32_t  ordinal,
gnsdk_link_data_type_t *  p_buffer_data_type,
gnsdk_byte_t **  p_buffer,
gnsdk_size_t *  p_buffer_size 
)

Retrieves content data that is described by the Link query handle options and data type flag.

Parameters
link_query_handle[in] Link query handle to retrieve content for
content_type[in] Type of content to request. Request cover art when retrieving album or video Products artwork. Request an image when retrieving Contributors, Works, Seasons, or Series content.
ordinal[in] Nth item of content_type to retrieve
p_buffer_data_type[out] Pointer to receive the content data type
p_buffer[out] Pointer to receive the buffer that contains the requested content
p_buffer_size[out] Pointer to receive the memory size pointed to by p_buffer
Returns
LINKERR_NotInited - gnsdk_link_initialize was not successfully initiated
LINKERR_HandleObjectInvalid - Given link query handle is not a valid handle
LINKERR_HandleObjectWrongType - Given link query handle is not a valid link query handle
LINKERR_InsufficientInputData - Insufficient input data
LINKWARN_NotFound - Requested content is not available
LINKERR_NoError - Content was successfully retrieved

Remarks:

Use this function to retrieve multiple pieces of content for a link query handle.

This API can be called repeatedly on the same link query handle with different content type requests.

Long Running Potential: Network I/O, File system I/O (for online query cache or local lookup)

GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_link_query_create ( gnsdk_user_handle_t  user_handle,
gnsdk_status_callback_fn  callback_fn,
gnsdk_void_t *  callback_data,
gnsdk_link_query_handle_t *  p_link_query_handle 
)

Creates a Link query handle. The application can create a single gnsdk_link_query_handle_t to query for multiple content items. You can create and operate multiple query handles that run simultaneously, however, each query handle must be used only by the original thread that created it.

Parameters
user_handle[in] User handle for the user requesting the query
callback_fn[in_opt] Callback function called by Link library to report status and progress
callback_data[in_opt] Data that is provided via calls to the callback function
p_link_query_handle[out] Pointer to receive the Link query handle
Returns
LINKERR_NotInited - gnsdk_link_initialize() was not successfully initiated
LINKERR_InvalidArg - Given user_handle or p_link_query_handle is null
LINKERR_HandleObjectInvalid - Given link query handle is not a valid handle
LINKERR_NoError - Link query handle was successfully created

GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_link_query_options_clear ( gnsdk_link_query_handle_t  link_query_handle)

Clears all options currently set for a given Link query handle.

Parameters
link_query_handle[in] Link query handle to clear options for
Returns
LINKERR_NotInited - gnsdk_link_initialize was not successfully initiated
LINKERR_HandleObjectInvalid - Given link query handle is not a valid handle
LINKERR_HandleObjectWrongType - Given link query handle is not a valid link query handle
LINKERR_NoError - Options successfully cleared

Remarks:

As Link query handles can be used to retrieve multiple enhanced data items, it may be appropriate to specify different options between data retrievals. You can use this function to clear all options before setting new ones.
