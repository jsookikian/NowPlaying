Macro Definition Documentation

#define GNSDK_DSP_FEATURE_QUALITY_DEFAULT

Indicates the returned Feature is properly generated and no specific qualities.

#define GNSDK_DSP_FEATURE_QUALITY_SHORT

Indicates the source audio was shorter than optimal, and therefore the quality of the generated DSP feature could be degredaded.

#define GNSDK_DSP_FEATURE_QUALITY_SILENT

Indicates the source audio is silent, and therefore the generated Feature is based on silense.

#define GNSDK_FP_FINGERPRINT_QUALITY_FLAG_DEFAULT   0x0000000

Indicates the fingerprint quality of the source audio has no special qualities.

#define GNSDK_FP_FINGERPRINT_QUALITY_FLAG_SHORT   0x0000001

Indicates the source audio is short, and therefore the generated fingerprint is also short. As of now, this only applies to CX fingerprints.

#define GNSDK_FP_FINGERPRINT_QUALITY_FLAG_SILENT   0x0000002

Indicates the source audio is silent, and therefore the generated fingerprint is silent. As of now, this only applies to CX fingerprints.
