GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_fp_string_free ( gnsdk_str_t  string)

Frees rendered fingerprint strings. Passing a null string value returns FPERR_NoError.

Parameters
string[in] Pointer to string buffer to free
Returns
FPERR_NotInited gnsdk_manager_initialize() was not successfully initiated
FPERR_HandleObjectInvalid Given string parameter is not a string value returned by GNSDK that should be freed
FPERR_NoError String was successfully freed

GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_handle_release ( gnsdk_handle_t  handle)

Decrement the reference count for the given handle. If the reference count goes to 0, all resource held by the handle will be freed, including the handle itself.

Parameters
handle[in] The handle whose reference count will be decremented.
GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_handle_report ( gnsdk_uintptr_t  client_ref)

Prints a report to stdout of all open handles. The report comprises the open handle count for each handle type with a non-zero count.

Parameters
client_ref[in] Specific client to report open handles for. For all clients (default) pass 0.
GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_manager_asset_fetch2 ( gnsdk_cstr_t  asset_url,
gnsdk_cstr_t  lookup_mode,
gnsdk_user_handle_t  user_handle,
gnsdk_status_callback_fn  callback_fn,
gnsdk_void_t *  callback_data,
gnsdk_byte_t **  pp_asset_data,
gnsdk_size_t *  p_asset_data_size,
gnsdk_cstr_t *  p_asset_data_type 
)

Experimental: Fetch a the raw asset data represented by the asset handle. This method can fetch data locally or from online depending on the provided URL.

Parameters
asset_url[in] Universal resource locator for asset to fetch
lookup_mode[in] the lookup mode to use to fetch the content asset. See the GNSDK_LOOKUP_MODE_* macros.
user_handle[in] GNSDK user handle
callback_fn[in_opt] Callback function called by manager library to report status and progress
callback_data[in_opt] Data that is provided via calls to the callback function
p_asset_data[out] asset data buffer (use gnsdk_manager_buffer_free() to free when done)
p_asset_data_size[out] asset data size
p_asset_data_type[out] asset data mime type
GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_manager_buffer_free ( gnsdk_byte_t *  p_buffer)

Frees the memory associated with an GNSDK provided buffer such as an asset buffer returned by gnsdk_manager_asset_fetch. Passing a null pointer value returns SDKMGRERR_NoError.

Parameters
p_buffer[in] Pointer to string buffer to free

Frees the memory associated with an GNSDK Manager-provided serialized string or XML string. Passing a null string value returns SDKMGRERR_NoError.
