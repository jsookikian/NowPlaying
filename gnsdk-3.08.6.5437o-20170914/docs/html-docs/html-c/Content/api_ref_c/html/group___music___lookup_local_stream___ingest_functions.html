Creates a local lookup stream ingest handle.

You need to create local lookup stream ingest handle before using other ingestion related APIs.

Parameters

callback_data[in_opt] Data that is passed back through calls to the callback
ph_ingest_handle[out] Pointer to receive the Local Lookup Stream Ingest handle
