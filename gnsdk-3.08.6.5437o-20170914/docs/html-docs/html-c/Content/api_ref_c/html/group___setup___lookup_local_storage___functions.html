GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_lookupdatabase_close ( gnsdk_cstr_t  db_identifier)

Close the specified lookup database - will wait if any threads have outstanding queries with this database. Once a lookup database is closed, any outstanding result GDOs from this database will be invalid. If this API is not called, all open databases will be closed on gnsdk_manager_shutdown()

Parameters
db_identifier[in] database identifer
GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_lookupdatabase_delete ( gnsdk_config_handle_t  config_handle)

Deletes the local database files. Note that only lookup related GDB files are deleted.

Parameters
config_handle[in] database configuration.
