Macro Definition Documentation

#define GNSDK_CONFIG_LOOKUPDATABASE_ACCESS

Allows for specifying whether the database should be open in read-only mode or read-write mode (which allows for the persistent storing of results).

#define GNSDK_CONFIG_LOOKUPDATABASE_ALL_LOCATION

Allows for specifying the directory for storing all of the database file(s)

#define GNSDK_CONFIG_LOOKUPDATABASE_CONTENT_LOCATION

Allows for specifying the directory for storing the content database file(s) which contain images (cover art, artist images, and so on).

#define GNSDK_CONFIG_LOOKUPDATABASE_ENABLE_EPG_RADIO

Allows for specifying that the database will be used for Radio Station queries and imagery. Performing Radio Station queries without enabling epg_radio will not result in an error, but Radio queries will return no results.

#define GNSDK_CONFIG_LOOKUPDATABASE_ENABLE_EPG_TV

Allows for specifying that the database will be used for TV Channel queries and imagery. Performing TV Channel queries without enabling epg_tv will not result in an error, but TV queries will return no results.

#define GNSDK_CONFIG_LOOKUPDATABASE_ENABLE_MUSICID

Allows for specifying that the database will be used for MusicID GDO queries.

#define GNSDK_CONFIG_LOOKUPDATABASE_ENABLE_MUSICID_CD

Allows for specifying that the database will be used for MusicID CD TOC Performing CD TOC queries without enabling musicid_cd will not result in an error, but TOC queries will return no results.

#define GNSDK_CONFIG_LOOKUPDATABASE_ENABLE_MUSICID_IMAGES

Allows for specifying that the database will be used to fetch MusicID imagery (cover art, artist images, and so on). Performing image queries without enabling musicid_images will not result in an error, but image queries will return no results.

#define GNSDK_CONFIG_LOOKUPDATABASE_ENABLE_MUSICID_TEXT

Allows for specifying that the database will be used for MusicID and/or MusicID-File text queries. Performing text queries without enabling musicid_text will not result in an error, but text queries will return no results.

#define GNSDK_CONFIG_LOOKUPDATABASE_EPG_LOCATION

Allows for specifying the directory for storing the electronic program guide database file(s) which contain metadata, indexes, and content related to radio stations and TV channels.

#define GNSDK_CONFIG_LOOKUPDATABASE_METADATA_LOCATION

Allows for specifying the directory for storing the metadata database file(s)

#define GNSDK_CONFIG_LOOKUPDATABASE_TEXTINDEX_LOCATION

Allows for specifying the directory for storing the text index database file(s)

#define GNSDK_CONFIG_LOOKUPDATABASE_TOCINDEX_LOCATION

Allows for specifying the directory for storing the CD TOC index database file(s)

#define GNSDK_LOOKUPDATABASE_ACCESS_READ_ONLY

Specifies that the database should be opened in read-only mode.

#define GNSDK_LOOKUPDATABASE_ACCESS_READ_WRITE

Specifies that the database should be opened in read-write mode, allowing for records and images to be persistently stored in this result database.
