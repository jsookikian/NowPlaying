GNSDK_EXPORTED gnsdk_cstr_t GNSDK_API gnsdk_lookup_local_get_build_date ( void  )

Retrieves the Lookup Local module's build date string. This API can be called at any time, even before initialization and after shutdown.

GNSDK_EXPORTED gnsdk_cstr_t GNSDK_API gnsdk_lookup_local_get_version ( void  )

Retrieves the Lookup Local module version string. This API can be called at any time, even before initialization and after shutdown.

GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_lookup_local_initialize ( gnsdk_manager_handle_t  sdkmgr_handle)

Initializes the Lookup Local module. This function must be successfully called before any calls to other Lookup Local APIs will succeed.

Parameters
sdkmgr_handle[in] Handle from a successful gnsdk_manager_initialize() call.
GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_lookup_local_shutdown ( void  )

Shuts down and release resources for the Lookup Local module. gnsdk_lookup_local_shutdown must be called to shutdown this module; all Lookup Local APIs cease to function until the module is again initialized.
