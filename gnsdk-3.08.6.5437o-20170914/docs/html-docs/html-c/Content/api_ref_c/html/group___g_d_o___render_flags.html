Macro Definition Documentation

#define GNSDK_GDO_RENDER_CREDITS

Renders any credit metadata into the output for a GDO.

#define GNSDK_GDO_RENDER_DEFAULT

Renders the default metadata into the output for a GDO (identifiers not included).

Remarks:

Equivalent to the set of flags:

  • GNSDK_GDO_RENDER_STANDARD
  • GNSDK_GDO_RENDER_GENRE_LEVEL2
#define GNSDK_GDO_RENDER_DESCRIPTORS

Renders all Gracenote List-based descriptors (genre, era, mood, etc...) values into output for a GDO.

#define GNSDK_GDO_RENDER_DISCOVER_SEEDS

Renders any Gracenote Discover Seed values into the output for a GDO.

Remarks:

Discover Seeds provide for compatibility with Gracenote Discover product. GDO render flags can be combined to output more metadata.

#define GNSDK_GDO_RENDER_FULL

Renders majority of metadata into the output for a GDO (identifiers not included)

Remarks:

Equivalent to the set of flags:

  • GNSDK_GDO_RENDER_DEFAULT
  • GNSDK_GDO_RENDER_CREDITS
  • GNSDK_GDO_RENDER_SORTABLE
  • GNSDK_GDO_RENDER_GNIDS
#define GNSDK_GDO_RENDER_GENRE_LEVEL1

Renders any Gracenote genres as level-1 (coarse) values into output for a GDO.

#define GNSDK_GDO_RENDER_GENRE_LEVEL2

Renders any Gracenote genres as level-2 (more granular) values into output for a GDO.

#define GNSDK_GDO_RENDER_GENRE_LEVEL3

Renders any Gracenote genres as level-3 (fine grained) values into output for a GDO.

#define GNSDK_GDO_RENDER_GNIDS
#define GNSDK_GDO_RENDER_GNUIDS
#define GNSDK_GDO_RENDER_JSON

Renders GDO metadata as JSON output (otherwise XML is rendered format)

Remarks:

Use this flag to render output as JSON. GDO render flags can be combined to output more metadata.

#define GNSDK_GDO_RENDER_NONEWLINES

Renders GDO without newlines (as single line of text)

Remarks:

#define GNSDK_GDO_RENDER_PRODUCT_IDS

Renders any Gracenote Product ID values into the output for a GDO.

Remarks:

Product IDs provide a static identifier for each type for which they are available. GDO render flags can be combined to output more metadata.

#define GNSDK_GDO_RENDER_RAW_TUIS

Renders any Gracenote TUI values into output for a GDO.

#define GNSDK_GDO_RENDER_SERIAL_GDOS

Renders any serialized GDO values into the output for a GDO.

Remarks:

Serialized GDOs provide an identifier that can be used to re-retrieve the respective type from Gracenote, or to use as input for follow-on queries on the originating type. GDO render flags can be combined to output more metadata.

#define GNSDK_GDO_RENDER_SORTABLE

Renders values for sorting names and titles into the output for a GDO. Note: this flag is going to be deprecated and sortable data will be rendered based on the presence of the '*_ENABLE_SORTABLE' query option on the initial query.

#define GNSDK_GDO_RENDER_STANDARD

Renders standard metadata into the output for a GDO.

Remarks:

Renders standard common metadata but without special identifiers: no credits and no Ids. GDO render flags can be combined to output more metadata.

#define GNSDK_GDO_RENDER_SUBMIT

Renders any Gracenote data supported for Submit editable GDOs into the output for a GDO. This rendered data includes both the supported editable and non-editable data. Use in conjuction with STANDARD, DEFAULT or FULL render flags.

#define GNSDK_GDO_RENDER_XML

Renders GDO metadata as XML output

Remarks:

As XML is the default output format, this flag just makes explicit that this API is to render the output as XML. GDO render flags can be combined to output more metadata.
