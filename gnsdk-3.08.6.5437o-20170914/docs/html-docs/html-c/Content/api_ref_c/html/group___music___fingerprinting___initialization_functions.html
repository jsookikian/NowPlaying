GNSDK_EXPORTED gnsdk_cstr_t GNSDK_API gnsdk_dsp_get_build_date ( void  )

Retrieves the DSP 's build date string for GNSDK.

Returns
Note Build date string of the format: YYYY-MM-DD hh:mm UTC

Remarks:

This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete. Example Example build date string: 2008-02-12 00:41 UTC

GNSDK_EXPORTED gnsdk_cstr_t GNSDK_API gnsdk_dsp_get_version ( void  )

Retrieves DSP's version string for GNSDK. This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete. Example: 1.2.3.123 (Major.Minor.Improvement.Build) Major: New functionality Minor: New or changed features Improvement: Improvements and fixes Build: Internal build number

GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_dsp_initialize ( gnsdk_manager_handle_t  sdkmgr_handle)

Initializes the DSP for GNSDK

Parameters
sdkmgr_handle[in] Handle from a successful gnsdk_manager_initialize call
Returns
DSPERR_InvalidArg Given sdkmgr_handle parameter is null
DSPERR_HandleObjectInvalid Given sdkmgr_handle is not a valid GNSDK Manager handle
DSPERR_InitFailed Initialization failed
DSPERR_NoError Initialization succeeded

Remarks:

This function must be successfully called before any other DSP APIs will succeed. This function can be called multiple times, however, all successful calls must be paired with a call to gnsdk_dsp_shutdown. If gnsdk_dsp_initialize returns an error, then gnsdk_dsp_shutdown must not be called.

GNSDK_EXPORTED gnsdk_error_t GNSDK_API gnsdk_dsp_shutdown ( void  )

Shut downs and releases resources for the DSP library.

Returns
DSPERR_NotInited gnsdk_dsp_initialize was not successfully initiated
DSPERR_NoError Shut down succeeded

Remarks:

All gnsdk_dsp_shutdown calls must be paired with an initial call to gnsdk_dsp_initialize. The final gnsdk_dsp_shutdown call results in the library being completely shut down; all DSP APIs cease to function until the library is again initialized.
