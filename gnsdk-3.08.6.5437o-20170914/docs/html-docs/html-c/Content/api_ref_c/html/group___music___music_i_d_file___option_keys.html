#define GNSDK_MUSICIDFILE_OPTION_BATCH_SIZE   "gnsdk_midf_option_batch_size"

Sets or retrieves the LibraryID batch size option.

Remarks:

When given a large number of files, LibraryID processes the files in batches. This option allows you to configure the size of the batches. The larger the batch, the fewer potential break-ups in any groupings; however, large batches require more resources to process the batch.

#define GNSDK_MUSICIDFILE_OPTION_ENABLE_CLASSICAL_DATA   "gnsdk_midf_option_enable_classical"

Indicates whether a response should include any associated classical music data.

Remarks:

To retrieve classical music data, define this option key with a True value for a specific MusicID-File query handle. This retrieves the data and automatically renders it into the XML output. An application's license must be entitled to retrieve this specialized data. Contact your Gracenote Professional Services representative with any questions about this enhanced functionality.

#define GNSDK_MUSICIDFILE_OPTION_ENABLE_CONTENT_DATA   "gnsdk_midf_option_enable_content"

Indicates whether a response should include data for use in fetching content (like images).

Remarks:

To allow the GDO to be efficiently used to retrieve content data, define this option key with a True value for a specific MusicID query handle. An application's client ID must be entitled to retrieve this specialized data. Contact your Gracenote Professional Services representative with any questions about this enhanced functionality.

#define GNSDK_MUSICIDFILE_OPTION_ENABLE_EXTERNAL_IDS   "gnsdk_midf_option_enable_xids"

Indicates whether a response should include any external IDs (third-party IDs).

Remarks:

External IDs are third-party IDs associated with the results (such as an Amazon ID), configured specifically for your application. An application's license must be entitled to retrieve this specialized data. Contact your Gracenote Professional Services representative with any questions about this enhanced functionality. External IDs are supported in the following types, for the listed child and value GDOs: GNSDK_GDO_TYPE_ALBUM GNSDK_GDO_CHILD_EXTERNAL_ID GNSDK_GDO_TYPE_TRACK GNSDK_GDO_CHILD_EXTERNAL_ID
