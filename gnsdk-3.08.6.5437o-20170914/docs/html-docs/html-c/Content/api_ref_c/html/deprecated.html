Global gnsdk_musicidstream_channel_automatic_set (gnsdk_musicidstream_channel_handle_t musicidstream_channel_handle, gnsdk_bool_t b_auto_set)
NB: Automatic mode for radio stream identification will be removed in future release. Do not use. Enables automatic identifications for a given MusicIDStream channel handle. This is only supported when the channel handle is created with the gnsdk_musicidstream_preset_radio value. The default mode when a channel is created is automatic = False.
Parameters
musicidstream_channel_handle[in] MusicIDStream channel handle to retrieve an option from
b_auto_set[in] True or False to set or unset automatic mode

NB: Automatic mode for radio stream identification will be removed in future release. Do not use. Returns the time to the next query, in milliseconds, when used as a key to gnsdk_musicidstream_channel_info_get() on a channel in automatic mode. This key does not apply to manual query mode.

Global GNSDK_MUSICIDSTREAM_OPTION_ENABLE_LINK_DATA

Will be deprecated in a future release. For content (like cover art, artist images, etc), use GNSDK_MUSICIDSTREAM_OPTION_ENABLE_CONTENT_DATA instead. For external 3rd party IDs, use GNSDK_MUSICIDSTREAM_OPTION_ENABLE_EXTERNAL_IDS instead.

Global GNSDK_MUSICIDSTREAM_OPTION_PREFERRED_LANG

Will be deprecated in a future release. The application should set the default locale, or to set a locale to the query/channel handle to indicate the preferred language the results should appear in. Developer should understand that this setting is only a preference. If the preferred language cannot be retrieved an alternate language result will be returned. Applications can inspect the language and script of the returned results to determine if the preference has been met.
