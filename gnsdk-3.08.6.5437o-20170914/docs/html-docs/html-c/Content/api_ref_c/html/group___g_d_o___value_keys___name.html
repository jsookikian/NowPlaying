Macro Definition Documentation

#define GNSDK_GDO_VALUE_DISPLAY

Retrieves the display value.

#define GNSDK_GDO_VALUE_DISPLAY_LANGUAGE

Retrieves the language of the display value as ISO code.

Note: Display language IDs are not available for track titles.

#define GNSDK_GDO_VALUE_DISPLAY_LANGUAGE_DISPLAY

Retrieves the display language as text (for example, "English").

#define GNSDK_GDO_VALUE_DISPLAY_SCRIPT

Retrieves the script used by the display values as ISO 15924 code

#define GNSDK_GDO_VALUE_FAMILY

Retrieves a family (or last) name value of a person from a GDO.

#define GNSDK_GDO_VALUE_GIVEN

Retrieves the given name for the display value.

#define GNSDK_GDO_VALUE_PREFIX

Retrieves the prefix of the display value.

#define GNSDK_GDO_VALUE_SORTABLE

Retrieves sortable text for a GDO, e.g. a name or a title. This key should only be used for sorting use case only
