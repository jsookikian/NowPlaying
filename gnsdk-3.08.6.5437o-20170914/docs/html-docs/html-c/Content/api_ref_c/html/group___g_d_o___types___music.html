#define GNSDK_GDO_CHILD_MATCH

This key allows you to iterate over the "matches" returned by gnsdk_musicid_query_find_matches(). For each match that is returned, you can inspect its type to see what kind of a GDO you have.

#define GNSDK_GDO_TYPE_ALBUM

GDO is of album type.

Remarks:

Value is used as return value for gnsdk_manager_gdo_get_type. Album types are retrieved from GNSDK_GDO_TYPE_RESPONSE_ALBUM types through gnsdk_manager_gdo_child_get with the GNSDK_GDO_CHILD_ALBUM key. This type supports Link data.

#define GNSDK_GDO_TYPE_AUDIO_WORK

GDO is of an audio work type.

#define GNSDK_GDO_TYPE_MATCH_INFO
#define GNSDK_GDO_TYPE_RESPONSE_ALBUM

GDO is of album response type.

Remarks:

Value is used as return value for gnsdk_manager_gdo_get_type. Album response types are returned from query APIs of GNSDKs that return album content. An album response type contains one or more GNSDK_GDO_TYPE_ALBUM child types.

#define GNSDK_GDO_TYPE_RESPONSE_MATCH

The type for GDOs returned from calls to gnsdk_musicid_query_find_matches(). Indicates that the GDO is a response object containing 0-n "match" children, which can be of various types.

#define GNSDK_GDO_TYPE_RESPONSE_MATCH_INFO
#define GNSDK_GDO_TYPE_RESPONSE_TRACK

GDO is of track response type.

#define GNSDK_GDO_TYPE_TRACK

GDO is of track type.

Remarks:

Value is used as return value for gnsdk_manager_gdo_get_type. Track types are retrieved from GNSDK_GDO_TYPE_ALBUM types through gnsdk_manager_gdo_child_get with one of the GNSDK_GDO_CHILD_TRACK keys. This type supports Link data.
