Macro Definition Documentation

#define GNSDK_GDO_VALUE_GLOBALID 
Retrieves a global id value from a GDO.
#define GNSDK_GDO_VALUE_GNID 
Retrieves a GNID value for a product from a GDO.
#define GNSDK_GDO_VALUE_GNUID 
Retrieves a GNUID value for a product from a GDO.
#define GNSDK_GDO_VALUE_PRODUCTID

Retrieves a product ID for the current product from a GDO.

Remarks:

Available for most types, this retrieves a value which can be stored or transmitted. This value can be used as a static identifier for the current content as it will not change over time. As this is a unique value, the ordinal parameter for gnsdk_manager_gdo_value_get must be set to 1 to retrieve the first (or only) value instance.

#define GNSDK_GDO_VALUE_RESPONSE_RESULT_COUNT 
Retrieves number of results in a RESPONSE GDO
#define GNSDK_GDO_VALUE_SPOKEN_LANGUAGE_ID

Retrieves a spoken language id value from a GDO. This value is just the spoken langauge id that is not currently assosiated with any list string.

#define GNSDK_GDO_VALUE_TAGID

Retrieves a TAGID value for a product from a GDO (Tag ID is a synonym for Product ID).

Remarks:

This value key exists primarily to support legacy implementations. We recommend using GNSDK_GDO_VALUE_PRODUCTID to retrieve product identifiers from a GDO type.

#define GNSDK_GDO_VALUE_TUI

Retrieves a TUI value from a GDO.

#define GNSDK_GDO_VALUE_TUI_MATCH_PRODUCT

Retrieves a second TUI from a Works GDO, if that GDO can be accessed by two IDs. This TUI is used for matching partial Products objects to full Works objects. Use this value to ensure correct TUI value matching for cases when a video Product GDO contains multiple partial Work GDOs. Each partial Work GDO corresponds to a full Works object, and each full Works object contains the GNSDK_GDO_VALUE_TUI value, incremented by one digit to maintain data integrity with Gracenote Service. The GNSDK_GDO_VALUE_TUI_MATCH_PRODUCT maps the partial Works object TUI value to the full Works object.

#define GNSDK_GDO_VALUE_TUI_TAG

Retrieves a TUI Tag value from a GDO.
