Detailed Description

Provides services for audio recognition using CD TOC-based search, text-based search, fingerprint, and identifier lookup functionality.

Retrieves the MusicID library version string.

Returns
gnsdk_cstr_t Version string, if successful

Remarks:

You can call this API after successfully instantiating a GnManager instance. The returned string is a constant. Do not attempt to modify or delete.

Example version string: 1.2.3.123 (Major.Minor.Improvement.Build)

Major: New functionality

Minor: New or changed features

Improvement: Improvements and fixes

Build: Internal build number
