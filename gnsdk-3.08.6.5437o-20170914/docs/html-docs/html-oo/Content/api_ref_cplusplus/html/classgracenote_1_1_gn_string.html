static GnString gracenote::GnString::manage ( gnsdk_str_t  gnsdk_managed_string)
inlinestatic

Internally used factory for special SDK-managed strings. Do not use for strings that do not originate from the GNSDK libraries. This will assume ownership of the SDK-managed string, i.e. the refcount will not be incremented. It is not necessary to release the string manually.
