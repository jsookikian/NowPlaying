Detailed Description

Experimental: Fetch asset raw data. The raw data is fetched during object construction and may result in lengthy network access if accessing from online.

Perform the fetch of the data from the provided URL. This method can fetch data from any valid URL. This class must be used to fetch data from a GNSDK local URL. Construction may result in lengthy network access if the URL is online and the target data is large.
