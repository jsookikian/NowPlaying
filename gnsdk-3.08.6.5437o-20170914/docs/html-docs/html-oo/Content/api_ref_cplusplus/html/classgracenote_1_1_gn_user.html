Reconstitutes user from serialized user handle data. Use this constructor to reconstitute a previously serialized GnUser. Reconstitution does not count towards the user count for your client in Gracenote Service. User's registered via Gracenote's Web API can also be used, this prevents an application that uses both GNSDK and Web API from having to register two users. To use a user from Web API construct it according to the following format and provide: <clientid>-<clienttag>:<userid>-<usertag>

Parameters
serializedUser[in] String of serialized user handle data
clientIdTest[in_opt] Serialized user's expected Client ID
